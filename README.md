# Emotion Vision Ultimate

Система распознавания эмоций в реальном времени с использованием компьютерного зрения и нейронных сетей.

Описание:

Программа анализирует видеопоток с веб-камеры, определяет лицо человека и классифицирует эмоциональное состояние по 7 категориям: радость, грусть, злость, удивление, страх, отвращение и нейтральное состояние.

Возможности:

-Распознавание 7 базовых эмоций в реальном времени

-Визуализация результатов с графиком истории

-Запись видео со статистикой эмоций

-Сохранение скриншотов

-Поддержка двух детекторов: FER (высокая точность) и DeepFace (универсальность)

# Быстрая установка с GitHub
(Bash команды)
1. Клонирование репозитория

git clone https://github.com/dzm1trok/EMOVISION.git


2. Установка зависимостей 


pip install opencv-python numpy

pip install fer

3. Запуск

python app.py

Требования для установки:

Python 3.8+

Веб-камера

Windows / Linux / macOS
